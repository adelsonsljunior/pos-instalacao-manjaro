## Como rodar

### 1. Clone o repositório

~~~
$ git clone https://github.com/adelsonsljunior/pos-instalacao-manjaro.git
~~~

### 2. Entre no diretório

~~~
$ cd pos-instalacao-manjaro/
~~~

### 3. Dê permissão de execução ao script

~~~
$ chmod +x script.sh
~~~

### 4. Execute o script

~~~
$ ./script.sh
~~~
